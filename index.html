Mis habilidades

Poseo la capacidad y experiencia necesarias para desarrollar estratégicamente ramas dentro de un proyecto, facilitando así una gestión eficiente del flujo de trabajo. Mi destreza incluye la habilidad para cambiar entre estas ramas con fluidez.
Poseo conocimientos fundamentales en PHP, lo que me permite desarrollar aplicaciones web dinámicas y eficientes. Mi experiencia incluye la capacidad de escribir y mantener scripts PHP, interactuar con bases de datos MySQL, y utilizar técnicas básicas de manejo de formularios y validación de datos.
Poseo conocimientos básicos en Figma, una herramienta de diseño de interfaz de usuario y prototipado. Mi experiencia incluye la creación y edición de diseños de alta fidelidad, el desarrollo de prototipos interactivos y la colaboración en tiempo real con equipos de diseño y desarrollo.
Mi experiencia abarca la creación y gestión de repositorios, la creación de ramas para el desarrollo de nuevas características o la corrección de errores, y la fusión de cambios entre ramas de manera eficiente. Además, estoy familiarizado con conceptos fundamentales de Git, como commits, merges y conflictos,
Mi experiencia abarca la comprensión de los fundamentos de JavaScript, incluyendo variables, tipos de datos, estructuras de control, funciones y manipulación del DOM. Además, tengo la capacidad de escribir scripts simples para mejorar la interactividad y la funcionalidad de las páginas web, así como de trabajar con eventos para crear experiencias de usuario dinámicas.
